# Customer Management Portal

Welcome to the initial release of the Python CLI Project! This project provides a robust command-line interface to manage customers, orders, and products efficiently. Built with the `click` library, it offers a seamless and structured way to interact with your data.

## Features & Commands

The CLI is organized into three main groups: **Customers**, **Orders**, and **Products**. Below is a detailed list of all available commands and their descriptions.

### 👥 Customers Command Group
Commands for managing customers.

| Command | Description | Options / Arguments |
| :--- | :--- | :--- |
| `add` | Adds a new customer by name to the database. | `<name>` (required) |
| `list` | Shows a list of all saved customers. | None |
| `summary` | Displays a total summary of each customer's spending and order categories. | None |
| `filter-by-product` | Finds and lists all customers that have purchased the specified product. | `<product_name>` (required) |
| `group-by-spending` | Analyzes customer spending and categorizes them into spending tiers (High, Moderate, Low). | None |
| `top-by-spending` | Retrieves the top 3 customers sorted by their lifetime spending. | `--order-by` (A for ascending, D for descending, default: D) |
| `filter-by-multiple-categories` | Finds customers who have placed orders across multiple distinct categories. | `--category-names` (Comma separated list) |

### 📦 Orders Command Group
Commands for managing orders.

| Command | Description | Options / Arguments |
| :--- | :--- | :--- |
| `add` | Adds a new order (product, price, category) to a specific customer. | `<customer_name>` (required), `<order>` (tuple: product, price, category) |

### 🛒 Products Command Group
Commands for managing and viewing products and its categories.

| Command | Description | Options / Arguments |
| :--- | :--- | :--- |
| `list-categories` | Lists all the unique categories based on past orders. | None |
| `revenue-per-category` | Calculates and displays the total revenue generated by each category. | None |
| `list` | Shows a list of all unique products that have been ordered. | None |

## Usage Examples

Below are practical examples of how to run every available command.

### Customers
```bash
# Add a new customer
python main.py customers add "John Doe"

# List all customers
python main.py customers list

# View a summary of all customers' spending and categories
python main.py customers summary

# Filter customers by a specific product
python main.py customers filter-by-product "Laptop"

# Group customers into spending tiers
python main.py customers group-by-spending

# Get the top 3 customers by spending (Ascending order)
python main.py customers top-by-spending --order-by A

# Filter customers who bought across multiple distinct categories
python main.py customers filter-by-multiple-categories --category-names "Electronics, Office Supplies"
```

### Orders
```bash
# Add an order for a customer (Product, Price, Category)
python main.py orders add "John Doe" "Laptop" 1200.00 "Electronics"
```

### Products
```bash
# List all unique categories
python main.py products list-categories

# Calculate and show total revenue per category
python main.py products revenue-per-category

# List all unique products ordered
python main.py products list
```
